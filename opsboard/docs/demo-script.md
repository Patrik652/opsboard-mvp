# Opsboard Demo Script (90 seconds)

1) **Landing (5s)**
   - “Opsboard unifies work execution and reliability.”
   - Click **Try demo account** if Firebase is configured.

2) **Boards (20s)**
   - Show board columns + cards.
   - “We can run ops tasks like a Trello board.”

3) **Incidents (20s)**
   - Show incident list with severity and state.
   - “Incidents and response state live here.”

4) **Status (10s)**
   - “Public‑style status view for services.”

5) **Audit (10s)**
   - Show timeline of actions.
   - “Every action is captured for compliance.”

6) **Analytics (10s)**
   - “Reliability pulse: uptime, open incidents, active cards.”

7) **AI Panel (15s)**
   - Select incident and click **Run workflow**
   - Show risk score, summary, recommended actions, and agent trace.

8) **Operations (10s)**
   - Record synthetic alert and create recovery snapshot.
   - “This is the production-readiness overlay: monitoring + audit + DR.”

Finish: “MVP shows how AI‑first dev can ship a professional ops system fast.”
